# 🎉 Çekiliş Uygulaması

Python ile geliştirilmiş basit bir terminal tabanlı çekiliş uygulamasıdır. Kullanıcı ekleme, listeleme, karıştırma ve rastgele kazanan seçme işlemlerini gerçekleştirebilir.

## 🚀 Özellikler

* 👤 Kullanıcı ekleme
* 📋 Kullanıcı listesini görüntüleme
* 🔀 Kullanıcı listesini karıştırma
* 🎲 Rastgele bir veya birden fazla kullanıcı seçme

## 🛠️ Kullanılan Teknolojiler

* Python 3
* `random` modülü
* `time` modülü

## 📂 Proje Yapısı

```
cekilis-uygulamasi/
│
├── main.py
└── README.md
```

## ▶️ Çalıştırma

1. Projeyi bilgisayarınıza indirin veya klonlayın.

```bash
git clone https://github.com/cerenaydiin/-cekilis--uygulamasi-.git
```

2. Proje klasörüne girin.

```bash
cd -cekilis--uygulamasi-
```

3. Programı çalıştırın.

```bash
python main.py
```

## 📸 Uygulama Özellikleri

Program açıldığında aşağıdaki işlemler yapılabilir:

* Kullanıcı ekleme
* Kullanıcıları görüntüleme
* Kullanıcı listesini karıştırma
* Rastgele çekiliş yapma

## 💡 Geliştirme Fikirleri

* Kullanıcı silme özelliği
* Dosyaya kayıt ve dosyadan okuma
* Çekiliş geçmişini görüntüleme
* Grafik arayüz (Tkinter veya PyQt)
* Aynı isimlerin tekrar eklenmesini engelleme
